## Javascript 변수 : var

https://www.w3schools.com/js/js_variables.asp

- 변수를 정의/선언할때 사용하는 키워드(예약어)
- ES5 버전에서 사용

```
var a; // 변수 하나씩 선언
var b;

var a, b; //한번에 여러 변수를 선언할 수 있다.
```
